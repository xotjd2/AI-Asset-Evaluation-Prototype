# AI 기반 금융상품 평가 프로토타입 기획서

## 1. 프로젝트 개요

본 프로젝트는 주식, 채권, 프로젝트 투자안을 하나의 화면에서 평가하고 결과를 저장할 수 있는 금융 의사결정 지원 프로토타입이다. 단순 수익성 계산에 그치지 않고 상품 유형별로 다른 위험지표를 함께 제시해 비교 가능한 판단 보조 도구를 만드는 것을 목표로 한다.

## 2. 기획 배경

- 실무에서는 평가 대상마다 계산 방식이 달라 엑셀 수작업이 반복된다.
- 수익성만 보고 판단하면 변동성, 금리 민감도, 부도확률 같은 핵심 리스크가 빠지기 쉽다.
- 결과를 한 번 보고 끝내지 않고 최근 평가 이력을 축적할 필요가 있다.

## 3. 목표

- 주식, 채권, 프로젝트를 하나의 대시보드에서 평가한다.
- 상품 유형별 핵심 위험지표를 자동 계산한다.
- 결과를 설명 문구와 함께 제시해 비전문가도 해석 가능하게 한다.
- 최근 평가 결과를 DB에 저장하고 재확인할 수 있게 한다.
- 웹 배포 기준으로 즉시 시연 가능한 MVP를 완성한다.

## 4. 대상 사용자

- 금융상품 초안 검토가 필요한 팀원
- 투자안 비교가 필요한 기획 및 운영 담당자
- 발표 또는 시연용 프로토타입이 필요한 프로젝트 팀

## 5. 핵심 기능

### 5.1 주식 평가

- 기대 연수익률과 변동성 기반 기대가치 계산
- `VaR 95%` 제시
- 위험점수와 위험등급 산출
- 상세 보기에서 민감도 분석 제공

### 5.2 채권 평가

- 할인현금흐름으로 채권가치 계산
- 수정 듀레이션, 컨벡서티 제공
- 금리 125bp 충격 시 손실 추정
- 상세 보기에서 금리 변화 시 가격 민감도 제공

### 5.3 프로젝트 평가

- NPV, IRR, 회수기간 계산
- 부도확률과 현재가치를 반영한 신용위험추정치 제공
- 할인율 및 부도확률 변화에 따른 민감도 분석 제공

### 5.4 평가 이력 관리

- 결과 저장 버튼으로 DB 적재
- 최근 10건 조회
- 평가 유형별로 구분된 최근 이력 표시

## 6. 차별점

- 서로 다른 자산 유형을 하나의 인터페이스에서 비교 가능한 형태로 제공한다.
- 수치만 보여주지 않고 계산 설명, 해석 코멘트, 핵심 하이라이트를 함께 제시한다.
- 결과 상세 보기에서 입력 요약과 민감도 분석을 동시에 제공한다.

## 7. 화면 구성

- 상단 히어로 영역: 서비스 목적과 사용 흐름 안내
- 입력 영역: 주식, 채권, 프로젝트 평가 폼
- 결과 영역: 평가 결과 카드, 위험등급, 상세 보기, 저장 버튼
- 최근 이력 영역: 최근 저장 결과를 유형별 컬럼으로 표시

## 8. 기대 효과

- 평가 시간 단축
- 리스크 중심 의사결정 보조
- 반복 시연과 발표에 적합한 웹 프로토타입 확보
- 금융 계산 로직과 웹 서비스 구현을 결합한 포트폴리오 결과물 확보

## 9. 배포 운영 기준

- 프론트엔드: Vercel
- 백엔드: Render
- 백엔드 API 주소: `https://ai-asset-evaluation-prototype.onrender.com`
- 프론트 환경변수: `VITE_API_BASE_URL`

## 10. 향후 개선 방향

- 사용자 인증 및 권한 관리
- 저장 결과 필터링과 검색
- PDF/엑셀 리포트 출력
- 시나리오 저장 및 비교
- 서버 측 재계산 검증과 자동 테스트 강화
