# AI 기반 금융상품 및 프로젝트 평가 시스템 기획서

## 1. 프로젝트 개요

본 시스템은 금융상품과 프로젝트 사업성 평가를 빠르게 수행하고, 주요 리스크 지표를 자동 산출하여 의사결정을 지원하는 웹 프로토타입이다. AI 도구를 활용해 프론트엔드, 백엔드, 문서 초안을 빠르게 생성하고, 계산 로직과 입력 검증은 별도로 검토하는 방식으로 구현한다.

## 2. 문제 정의

- 가치평가와 리스크 측정이 엑셀 중심으로 이뤄지면 반복 작업이 많다.
- 평가 기준이 사람마다 달라져 결과 일관성이 떨어질 수 있다.
- 금융상품과 프로젝트를 같은 화면에서 비교하기 어렵다.

## 3. 목표

- 채권과 프로젝트를 대상으로 한 평가 MVP 구축
- VaR, Duration, Convexity, NPV, IRR 등 핵심 지표 자동화
- 평가 결과를 요약 의견과 함께 시각적으로 제공
- 평가 이력을 DB에 저장해 반복 검토 가능하게 구성

## 4. 핵심 사용자

- 자산운용 담당자
- 리스크 관리 담당자
- 투자 심사역

## 5. 핵심 기능

- 채권 입력 및 가치평가
- 프로젝트 현금흐름 입력 및 사업성 평가
- 내부 리스크 점수와 등급 산출
- 최근 평가 이력 조회

## 6. 기대효과

- 입력부터 결과 확인까지의 평가 시간을 단축
- 금융 지표와 내부 점수 체계를 결합해 일관된 판단 지원
- AI 기반 개발 생산성과 검증 역량을 동시에 보여줄 수 있음
