# 개발문서

## 1. 기술 스택

- Frontend: React, Vite
- Backend: Spring Boot, Spring Web, Spring Data JPA
- Database: MySQL

## 2. 아키텍처

- React 대시보드가 사용자 입력을 수집
- Spring Boot REST API가 평가 로직 수행
- 결과를 MySQL에 저장 후 응답 반환
- 최근 평가 이력을 다시 조회하여 화면에 표시

## 3. 도메인 모델

### evaluation_records

- `id`
- `evaluation_type`
- `target_name`
- `present_value`
- `var95`
- `duration_value`
- `convexity`
- `npv`
- `irr`
- `payback_period`
- `risk_grade`
- `summary`
- `created_at`

## 4. API 명세

### POST /api/evaluations/bond

입력값
- `bondName`
- `faceValue`
- `couponRate`
- `marketYield`
- `maturityYears`
- `confidenceLevel`

출력값
- 현재가치
- VaR 95%
- Duration
- Convexity
- 리스크 점수 및 등급

### POST /api/evaluations/project

입력값
- `projectName`
- `initialInvestment`
- `discountRate`
- `probabilityOfDefault`
- `cashFlows`

출력값
- NPV
- IRR
- 회수기간
- VaR 95%
- 리스크 점수 및 등급

### GET /api/evaluations/recent

- 최근 10건 평가 이력 반환

## 5. AI 활용 및 검증

- AI 활용
  - UI 초안 생성
  - REST API 구조 생성
  - 계산 로직 템플릿 작성
  - 문서 초안 작성

- 검증
  - 채권 현재가치, Duration, Convexity 공식 수기 검토
  - 프로젝트 NPV, IRR, 회수기간 결과 확인
  - 음수 및 빈 입력에 대한 예외 처리 점검
  - 프론트와 백엔드 간 인터페이스 필드명 일치 확인
